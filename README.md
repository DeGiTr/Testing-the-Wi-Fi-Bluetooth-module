Инструкция (RU)
Назначение: проверка работоспособности Wi‑Fi и Bluetooth адаптеров M.2 без измерений качества.

Как запустить:

Запустите файл run_wifi_bluetooth_test.bat двойным кликом.
Файл можно запускать из любого места (диск, рабочий стол, флешка).
После запуска откроется окно, появится приветствие и итог теста.
Окно автоматически закроется через ~8 секунд.
Результаты:

Зелёный текст Тестирование успешно! — Wi‑Fi и Bluetooth в норме.
Красный текст Тестирование неуспешно — хотя бы один из двух не в норме.
Требования:

Windows 10/11.
Права пользователя достаточно обычные.
Instruction (EN)
Purpose: basic functionality check for Wi‑Fi and Bluetooth M.2 adapters (no performance measurements).

How to run:

Double‑click run_wifi_bluetooth_test.bat.
You can run it from any location (disk, desktop, USB drive).
A window opens, shows the welcome line and test result.
The window closes automatically after ~8 seconds.
Results:

Green Тестирование успешно! — Wi‑Fi and Bluetooth are OK.
Red Тестирование неуспешно — at least one of them is not OK.
Requirements:

Windows 10/11.
Standard user permissions are sufficient.
